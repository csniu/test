# Challenge Code Frontend

这个 Challenge Code 项目的前端部分，基于 [vue-vben-admin v5.6.0](https://github.com/vbenjs/vue-vben-admin/releases/tag/v5.6.0) 版本进行开发。

## Install dependencies

```bash
cd vue-vben-admin
npm i -g corepack
pnpm install
```

## Run

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
